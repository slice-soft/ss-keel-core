We support only the latest released version of Keel. We recommend always keeping your dependency up to date.

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

If you discover a security issue, please report it privately by emailing:

**slicesoft@juancadev.com**

Include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix if you have one

We will acknowledge your report within **48 hours** and aim to release a fix within **7 days** depending on severity.

We appreciate responsible disclosure and will credit you in the release notes unless you prefer to remain anonymous.
